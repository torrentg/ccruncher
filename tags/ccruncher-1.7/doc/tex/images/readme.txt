

gpi files
--------------------------

gnuplot instructions files
use gnuplot to process it
gnuplot
  > load "filename.gpi"
  > quit


svg files
--------------------------

Scalable Vector Graphics files
use inkscape or other svg editor to open it


ps + eps + epsi files
--------------------------
Postscript files
Encapsulated Postscript files
Encapsulated Postscript Interchange files
use ghostview (gv) to open it

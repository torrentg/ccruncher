
zlib library installation
--------------------------------------------------

Linux systems:
  - install zlib packages
    [apt-get|yum] install zlib zlib-devel

Windows systems:
  - download the zlib windows compiled DLL from:
    http://www.zlib.net/
  - unpack it in this directory ($CCRUCNHER/deps/zlib-xxx)
  - copy zlib1.dll in $CCRUCNHER/bin/

Other systems:
  - download package from:
    http://www.zlib.net/
  - follow installation instructions

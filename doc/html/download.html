Download

Please, before download, read requeriments section. After download and unpack, read the file README.

Binary release. With this option you don't need to compile nothing. It is ready to use. Source code not included. Only for Linux environement.
  gunzip ccruncher-0.2_bin.tar.gz
  tar -xvf ccruncher-0.2_bin.tar

Source release. This package contains all elements for create the application. You need to download external libraries (see requeriments section) and compile the code.
  gunzip ccruncher-0.2.tar.gz
  tar -xvf ccruncher-0.2.tar

Repository. You can obtain last version from repository. You need a subversion client that you can download from subversion.tigris.org. This option is risky since current repository version can be unstable.
  svn checkout http://www.generacio.com/svn/repos/ccruncher/trunk/

SourceForge. CreditCruncher is under the SourceForge umbrella. You can obtain all oficial releases from SourceForge servers.

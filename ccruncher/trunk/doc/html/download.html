Download

Recuerde que la instalación de CreditCruncher depende de la existencia de otros elementos (pe. librería xerces-c). Lea detenidamente las instrucciones de instalación y el fichero README que acompaña a la distribución.

Binary release. No requiere realizar ningún paso de compilación ni instalación posterior. Contiene todo lo necesario para la ejecución de la aplicación pero no contiene el código fuente.
  gunzip ccruncher-0.1_bin.tar.gz
  tar -xvf ccruncher-0.1_bin.tar

Source release. Contiene todos los elementos para la creación de la aplicación CreditCruncher. Requiere seguir los pasos de instalación para generar los ejecutables e instalarlos.
  gunzip ccruncher-0.1.tar.gz
  tar -xvf ccruncher-0.1.tar

Repository. Puede obtener la última versión de los elementos que componen la aplicación CreditCruncher. Esta opción requiere tener instalado el cliente de Subversion.
  svn checkout http://www.generacio.com/svn/repos/ccruncher/trunk/

SourceForge. CreditCruncher se encuentra publicado en SourceForge. Puede obtener todas las versiones oficiales.

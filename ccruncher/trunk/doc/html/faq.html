CreditCruncher - FAQ

1. What is the project maturity ?

The main goal of CreditCruncher is compute the VAR for a large credit risk portfolio (about 50.000 clients grouped in some sectors) using MonteCarlo. At this moment (version 0.3 alpha), the main goal is not yet achieved. The reasons are:

  • The portfolio price algorithm need to be reviewed (not critical)
  • The copula generator uses an algorithm based on a matrix NxN. A matrix of 50.000x50.000 elements is not acceptable. Even so, works well for some hundreds of clients.
  • Documentation is under construction (this works slow)
  • The parallel procesing code is not yet implemented

In short, CreditCruncher-0.3 have the main framework builded and produce outputs, but can't be used in a real environement, yet. Is in alpha developement stage.

2. Windows platform is suported ?

Yes, but have some know (technical) bugs. The main stream is in unix environement. Project compile fine using cygwin. Also, the source distribution include the project files for MSVC7 compiler. MSVC6 is not suported because embeded projects (MersenneTwister and TNT/Jama) have problems with this compiler due to the obsolete implementation of C++ standar made by Microsoft (variable scope is non ISO C++ standar).

3. CreditCruncher have a graphic environement ?

No. CreditCruncher has been designed to work in batch mode, without graphical support. The output are plain text files containing the portfolio values obtained in each simulation. In adition, CreditCruncher supply utilities for a report generation (graphics, etc.) using unix utilities (shell scripts, gnuplot). If the user desire make this work, he can take output files and process with Excel to produce the distribution of portfolio value and compute VAR.

4. Can I found docs where the CreditCruncher algorithm is explained?

Yes. The document is named CreditCruncher - Technical Document and you can found a link here. This document is included in all distributions under the directory docs/. At this moment, it is under developement and only in spanish. It will be translated to english when completed.

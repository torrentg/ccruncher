Hardware Recomendations

En función del problema a resolver, CreditCruncher puede llegar a realizar muchas operaciones en coma flotante, usar una cantidad importante de memoria y crear ficheros de resultados de tamaño elevado. Por este motivo recomendamos la siguiente configuración mínima:

CPU Pentium IV 2400
RAM 512 Mb
Hard disk 10 Gb free

Esto no significa que la aplicación no funcione en máquinas que no cumplan los requerimientos de hardware mínimo recomendados. Estos pueden ser relajados si se pretende resolver problemas de tamaño pequeño o para máquinas que formen parte de un cluster (el tamaño de disco de los nodos esclavos puede ser reducido a algunos megas).

Xercesc libraries

CreditCruncher usa xercesc para realizar la interpretación del fichero de definición del problema. Instale las librerias dinámicas de xercesc según lo indicado en el site.

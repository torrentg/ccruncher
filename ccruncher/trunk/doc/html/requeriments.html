System Requeriments

List of environements where is expected that a CreditCruncher installation works without problems:

System flavour compiler glibc tested notes
Linux 2.6 Fedora Core 2 gcc 3.3.3 2.3.3 YES developement environement
Linux 2.6 Fedora Core 3 gcc 3.4.2 2.3.3 YES developement environement
linux other - - NO expat supported
AIX 5.1 - - NO expat supported
solaris 2.8 - - NO expat supported
hp ux 11i - - NO expat supported

If you make a instalation in a system not tested or listed, please, report it, explaining problems found, modifications realized, etc.

Expat library

CreditCruncher use expat for xml input file parsing. Expat is linked statically. If you take binary installation you don't need install expat library. Otherwhise, you need install the expat package from http://expat.sourceforge.net/, that include the libexpat.a and expat.h files. These elements are needed at compilation time.

lam-mpi environement

If you plan to deploy CreditCruncher in a cluster (to take advantage of parallelization capabilities) you need an LAM/MPI installation. Otherwise, forget this section and LAM/MPI.

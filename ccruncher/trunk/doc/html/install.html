CreditCruncher - Installation Instructions

If you make a instalation in a system not tested or not listed, please, report it, explaining problems found, modifications realized, etc.

Download

Select a distribution and download it.

Binary release. With this option you don't need to compile nothing. It is ready to use. Source code not included. Only for Linux environment.

Source release. This package contains all elements for create the application. You need to download external libraries (see installation section) and compile the code.

Repository. You can obtain last version from repository. You need a subversion client that you can download from subversion.tigris.org. This option is risky since current repository version can be unstable.
  svn checkout http://www.generacio.com/svn/repos/ccruncher/trunk/

Installation

Step 1. If you have select a packed distribution, unpack it.
  tar -xvzf ccruncher-X.Y_ZZZ.tgz
  cd ccruncher-X.Y

Step 2. CreditCruncher use library named expat. Check that you have the expat package installed (find file libexpat.so, usually in folder /usr/lib). If you have a source distribution check that you have the expat-devel package installed (find file expat.h, usually in folder /usr/include).

Step 3. If you have a source distribution, run next commands:
  ./configure --prefix=$PWD
  make
  make install

Usage

Step 1. Create a XML CreditCruncher input file. First, read Technical Document and Input File Reference. You can use a file from directory samples as template.

Step 2. Run ccruncher.
  bin/ccruncher --help
  bin/ccruncher -vf --hash=100 --path=./data file.xml

Step 3. Check generated output (plain text files). First field is the simulation counter, second field is the simulated segment value.
  ls -l data/*
  more data/segmentation-segment.out

Step 4. You can compute the VAR opening result files with a spreadsheet and computing the searched percentil. CreditCruncher have some tools that can help in your report generation. Try to run next commands:
  bin/plotdata -h
  bin/plotdata -p data/segmentation-segment.out
  bin/plotdata -c data/segmentation-segment.out
  bin/plotmtrans -h
  bin/plotmtrans -f file.xml
  bin/plotmtrans -c file.xml
  bin/plotmtrans -s file.xml
